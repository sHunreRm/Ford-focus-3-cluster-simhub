Эмулятор приборной панели Ford Focus 3 через Arduino + MCP2515 для работы с SimHub. 

Поддерживаемые игры:
Beamng drive
Forza horizon 5
Asseto corsa

Поддерживаемые функции (Зависит от игры):
АБС
RPM
Скорость
температура двигателя
перегрев двигателя
уровень топлива
поворотники
сигнал аварийной остановки
